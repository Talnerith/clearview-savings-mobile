# assets

App icon and splash image go here (e.g. `icon.png`, `splash.png`), referenced
from `app.json`. Until custom art is added, Expo uses its defaults — no icon
config is set in `app.json` so the build does not fail on a missing file.

When adding art, keep the branding rules in `CLAUDE.md`: neutral palette, no
real-bank impersonation, nothing that reads as "demo"/"simulation" to a patient.
